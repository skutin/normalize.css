HTML5 native elements

This document represents probably all HTML5-compatible tags with their native behavior.

For example, this is a <header> element - a group of introductory or navigational aids. You can use it in any <section> on your site, as many times as you want. This one is inside a <main> root element.

Text-level semantics

<a> Link
<abbr> HTML
<b> Bold text.
<bdo> Reversed text
<cite> Citation
<code> if (true) { alert( array['key'] ); }
<dfn> Term
<em> Important!
<kbd> User input
<mark> Marked text
<q> Quote text, and also inner quote
<s> Text, no longer relevant
<samp> Result of script
<small> Small text
<span> Sample span
<strong> Strong text
<var> Variable
<wbr> Veryveryveryveryveryveryverylong-word
<sup>,<sub> Different combinations of these elements: like this text12, or this text12
<del> Deleted entity
<ins> Addition
<output> Result of calculation
<time> December, 21

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea.

A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.
